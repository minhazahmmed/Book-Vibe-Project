## 📌 Overview

**Book Vibe** is a personal book tracking web app where you can discover books, mark them as read, save them to your wishlist, and explore detailed information about each title — all in a clean, responsive interface.

---

## ✨ Key Features

- 📖 Browse a curated collection of books across multiple genres
- 🔖 Add books to your **Read List** and track your progress
- ❤️ Save books to your **Wishlist** for future reading
- ⭐ View **ratings & reviews** for every book
- 🔃 **Sort** your list by rating (High → Low / Low → High)
- 🔔 Beautiful **SweetAlert2** notifications for all actions
- 📱 Fully **responsive** across all devices
